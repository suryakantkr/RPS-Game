# Rock Paper Scissors Game ✊✋✌️

A clean, minimalist, and fully interactive frontend implementation of the classic **Rock Paper Scissors** game. This project features a modern UI with responsive elements, fluid hover states, and dynamically calculated score tracking built entirely from scratch.

---

## 🚀 Live Demo

Check out the live interactive preview here: **[https://rock-paper-scissors-vercel-app.vercel.app/]**

---

## ✨ Features

* **Interactive Control Board:** Visually distinct, color-coded choice buttons for Rock (Pink), Paper (Purple), and Scissors (Blue) wrapped with crisp vector SVG icons.
* **Smart Game Logic:** Completely automated computer opponent using pseudo-random generation to ensure fair, real-time gameplay.
* **Live Scoreboard Tracker:** Instantly updates and retains the user's score vs. the computer's score dynamically during the session without reloading.
* **Context-Aware Status Bar:** The primary bottom pill banner acts as a dynamic messaging system, updating its text and color scheme natively based on game state (Neutral for a draw, Green for a win, and Red for a loss).
* **Hover Animations:** Micro-interactions and smooth scale transitions applied to choices to elevate user experience.

---

## 🛠️ Tech Stack

* **HTML5:** Structural markup utilizing semantic tags (`<header>`, `<main>`, `<button>`) for optimal web accessibility.
* **CSS3:** Custom styles managed through modern layouts (Flexbox alignment), Google Serif typography, and precise viewport scaling.
* **Vanilla JavaScript:** Event listeners, DOM manipulation, condition evaluation matrices, and state tracking logic without external dependencies.
